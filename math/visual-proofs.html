Visual Proofs

A visual proof or a proof without words is a proof of an identity or mathematical statement which can be demonstrated as self-evident by a diagram without any accompanying explanatory text. Such proofs can be considered more elegant than more formal and mathematically rigorous proofs due to their self-evident nature.

Examples
Pythagorean theorem 1 + 2 + ... + n = n (n + 1) / 2 1 + 3 + ... + (2n - 1) = n2 12 + 22 + ... + n2 = n (n + 1) (2n + 1) / 6 1/4 + 1/16 + 1/64 + 1/256 + ... = 1/3 1/3 + 1/9 + 1/27 + 1/81 + ... = 1/2 1/2 + 1/4 + 1/8 + 1/16 + ... = 1
